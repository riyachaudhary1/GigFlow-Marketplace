# GigFlow - Freelance Marketplace

## Overview
GigFlow is a full-stack platform where Clients can post jobs and Freelancers can bid on them. It includes secure authentication and a complete hiring workflow.

## Features
- **Authentication:** Login/Register (Client & Freelancer).
- **Jobs:** Post Gigs (Title, Description, Budget).
- **Bidding:** Freelancers can bid on open gigs.
- **Hiring Logic:** Clients can hire one freelancer, automatically rejecting others.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express + MongoDB

## How to Run
1. **Backend:**
   cd backend
   npm install
   npm start

2. **Frontend:**
   cd frontend
   npm install
   npm run dev
